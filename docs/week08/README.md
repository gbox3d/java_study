# Week 08 - 파일 I/O 및 직렬화

## 학습 목표
- 문자/버퍼 스트림 사용
- 파일 읽기/쓰기 실습
- 객체 직렬화와 transient 확인

## 예제 클래스
- 패키지: fileio
- 클래스: ex01, ex02, ex03

## 실행 방법

