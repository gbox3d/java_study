# Week 07 - 모던 자바

## 학습 목표
- 제네릭 타입 안정성 이해
- 람다식으로 코드 간결화
- 스트림 파이프라인 처리

## 예제 클래스
- 패키지: modernjava
- 클래스: ex01, ex02, ex03

## 실행 방법
### Windows
~~~pwsh
.\gradlew.bat :app:runWeek07Ex01
.\gradlew.bat :app:runWeek07Ex02
.\gradlew.bat :app:runWeek07Ex03
~~~

### macOS / Linux
~~~bash
./gradlew :app:runWeek07Ex01
./gradlew :app:runWeek07Ex02
./gradlew :app:runWeek07Ex03
~~~
