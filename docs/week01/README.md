# Week 01 - 자바 기초 및 개발 환경

## 학습 목표
- JVM, 컴파일(.java -> .class), 실행 흐름 이해
- 기본 입출력과 타입/변수 사용
- 제어문 반복문 작성 능력 확보

## 예제 클래스
- 패키지: basic
- 클래스: ex01, ex02, ex03

## 실행 방법
### Windows
~~~pwsh
.\gradlew.bat :app:runWeek01Ex01
.\gradlew.bat :app:runWeek01Ex02
.\gradlew.bat :app:runWeek01Ex03
~~~

### macOS / Linux
~~~bash
./gradlew :app:runWeek01Ex01
./gradlew :app:runWeek01Ex02
./gradlew :app:runWeek01Ex03
~~~
